# My Book Finder App

## Overview
My Book Finder App is an Android application that allows users to search for books using the Google Books API. Users can view book details, save books to their personal bookshelf, and manage their saved books.

## Features
- **Search for Books**: Users can search for books by title or author.
- **Book Details**: Users can view detailed information about the books including title, author, and a brief description.
- **Save to Bookshelf**: Users can save books to their personal bookshelf for later viewing.
- **View Bookshelf**: Users can view all the books they have saved.
- **Remove Books**: Users can remove books from their bookshelf.

## Technical Requirements
- Andriod Studio *Version*: Android Studio Koala Feature Drop | 2024.1.2 (What i used)
- **Android SDK**: Target SDK 34
- **Language**: Kotlin
- **Dependencies**:
  - Retrofit for network requests
  - Gson for JSON parsing
  - Room for database
  - Coroutine for asynchronous tasks
  - Glide for image loading
